# Ascended Fitness by Coach AP - Personal Training Website

## Overview
A comprehensive website for a personal training business incorporating the best features from leading platforms like TrueCoach, Trainerize, Mindbody, and ClubReady.

## Files Included
- `personal_training_website.html` - Main landing page with services, about, testimonials, and contact
- `client_portal.html` - Client portal with workout tracking, nutrition logging, progress tracking, and messaging
- `README.md` - This file

## Key Features Implemented

### From TrueCoach
- Workout generator with drag-and-drop functionality
- Video exercise library concept
- Client progression tracking
- Nutrition tracking interface
- Client communication hub
- Business management tools

### From Trainerize
- Training program delivery system
- Nutrition coaching tools
- Habit coaching features
- Client engagement tools
- Progress tracking dashboard

### From Mindbody
- Booking and scheduling system
- Payment processing integration
- Client management dashboard
- Marketing features

### From ClubReady
- Member experience tools
- Staff management concepts
- Reporting and analytics

## Design Features
